# getCryptocurrencyRate

仮想通貨のレートを取得するライブラリ<br>
bitflyer.com の WEBAPI を勝手に取得します<br>

# import

```python
from getCryptocurrencyRate import getCryptocurrencyRate
```

# use

```python
getCryptocurrencyRate("BTC_JPY", "hour").get() #取得する通貨 間隔
```
